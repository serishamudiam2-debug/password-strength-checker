# 🔐 Password Strength Checker
A Python-based tool that analyzes password strength and provides suggestions to improve security.

## 🚀 Features
- Detects weak, medium, and strong passwords
- Checks for:
  - Length
  - Uppercase & lowercase letters
  - Numbers
  - Special characters
- Provides suggestions to improve password strength
